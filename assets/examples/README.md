# Examples Guide

Папка для реальных примеров кадров, на которые ссылается мануал.

## Шаблон именования

`<section>-<scene>-<light>-<take>.jpg`

Примеры:

- `portrait-street-shade-01.jpg`
- `portrait-window-indoor-02.jpg`
- `night-city-signs-01.jpg`
- `night-cafe-warmlight-02.jpg`

## Минимальный набор для старта

- Портрет улица (2-3 кадра)
- Портрет у окна (2-3 кадра)
- Ночная улица с вывесками (2-3 кадра)
- Ночной портрет с фонарем/витриной (2-3 кадра)

## Правило

После добавления новых примеров обновлять соответствующий раздел в `guide/` ссылкой на файл и кратким комментарием, что в кадре получилось/не получилось.
