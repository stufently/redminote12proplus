# Examples Guide

Папка для реальных примеров кадров, на которые ссылается мануал.

## Шаблон именования

`<section>-<scene>-<light>-<take>.jpg`

Примеры:

- `portrait-street-shade-01.jpg`
- `selfie-outdoor-day-01.jpg`
- `food-flatlay-01.jpg`
- `group-friends-outdoor-01.jpg`

## Категории

| Папка | Содержимое | Кол-во |
|---|---|---|
| `portrait/` | Портреты (улица, окно, вечер, кафе) | 4 |
| `night/` | Ночные кадры (город, неон, следы света) | 4 |
| `selfie/` | Селфи (улица, окно, вечер, зеркало) | 4 |
| `food/` | Еда (flat lay, кафе, десерт, street food) | 4 |
| `group/` | Групповые (улица, кафе, праздник) | 3 |
| `ui/` | Скриншоты интерфейса камеры (пока пусто) | 0 |

## Источники

Все фото взяты из свободных фотобанков (Pexels, Unsplash, Pixabay) с лицензией, разрешающей свободное использование.

## Правило

После добавления новых примеров обновлять соответствующий раздел в `README.md` ссылкой на файл и кратким комментарием, что в кадре получилось/не получилось.
