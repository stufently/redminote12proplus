# 04. Портрет

## Быстрый сценарий

1. Выберите режим `Портрет`
2. Поставьте модель в мягкий свет (окно, тень, пасмурная улица)
3. Тапните по глазам для фокуса
4. Проверьте фон: не должен "резать" голову линиями и столбами
5. Снимите 5-10 кадров с разной дистанцией

## Позиционирование

- Камеру держите чуть выше уровня глаз для более аккуратной геометрии лица
- Не снимайте слишком близко, чтобы не искажать пропорции

## Что проверять после кадра

- Резкость глаз
- Естественность размытия по краям
- Цвет кожи без явного пересвета

## Если результат слабый

- Смените фон на более простой
- Поверните модель на 20-30 градусов к свету
- Сделайте серию в обычном режиме `Фото` для сравнения
