# Prompt Contract

The advisor prompt must enforce:

1. External second-opinion on planning quality
2. Explicit use of both pending and completed work
3. Concrete next actions with file references
4. Session-ready definition of done

The script must include these files when available:

- `workflow/BACKLOG.md`
- `workflow/CURRENT_STATE.md`
- `workflow/SESSION_LOG.md`
- `workflow/HANDOFF.md`
